# History_Blog_Site

this is a simple home for a fictional website called Herodote, focused on World History articles and developed with HMTL and CSS. The purpose of it was just for practicing Backend (HTML and CSS), therefore the project is static.
I may add Javascript features in the future to support a login system.


- To execute:	
  The most easy way is downloading and extracting the ZIP file. Then, open the Index HTML file in any browser.
